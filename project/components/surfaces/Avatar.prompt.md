Round avatar for staff, patients, testimonials.

```jsx
<Avatar src="patricia.jpg" alt="Patricia Nieto" size="lg" />
<Avatar name="Patricia Victoria" tone="mint" />
<Avatar size="md"><img src="assets/emblem-nieto.png" alt="Nieto"/></Avatar>
```

Falls back to initials from `name`. Sizes `xs`–`xl`; tones `sage` / `mint` / `gold` / `forest`.
