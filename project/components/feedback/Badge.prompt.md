Status pill for order states, labels, counts.

```jsx
<Badge variant="success" dot>Lista para retirar</Badge>
<Badge variant="warning" dot>En preparación</Badge>
<Badge variant="gold">Nuevo</Badge>
```

Variants: `neutral` (sage), `success`, `warning`, `danger`, `info`, `gold`, `solid` (forest). Add `dot` for a status indicator.
