Icon-only button for toolbars, dialogs, and social links. Always provide `label`.

```jsx
<IconButton label="Cerrar" variant="ghost"><XIcon/></IconButton>
<IconButton label="WhatsApp" variant="soft" shape="round"><MessageIcon/></IconButton>
```

Variants: `solid` (forest), `soft` (sage tint), `ghost`, `outline`. Sizes `sm`/`md`/`lg`. Use `shape="square"` for toolbar groups.
